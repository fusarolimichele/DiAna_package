# Get DiAna reference

This function provides the DiAna reference, that should be used when
using DiAna.

## Usage

``` r
DiAna_reference(print = TRUE)
```

## Arguments

- print:

  Logical, should the output be printed? Default to TRUE.

## Value

DiAna reference
