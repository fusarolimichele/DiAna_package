# Country Dictionary Dataset

A dataset containing mappings of country names, their abbreviations, and
their respective continents.

## Usage

``` r
country_dictionary
```

## Format

A data frame with 3 variables:

- countryname:

  Character. The name of the country, which can include official names
  and various abbreviations.

- country:

  Character. The official name of the country.

- continent:

  Character. The continent to which the country belongs.

## Details

This dataset provides a comprehensive mapping of country names and their
respective continents. It includes various forms of country names such
as official names and abbreviations. It is used internally for functions
such as descriptive.
